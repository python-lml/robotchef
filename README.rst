Installation
================================================================================

You can install it via pip:

.. code-block:: bash

    $ pip install robotchef


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/robotchef.git
    $ cd robotchef
    $ python setup.py install
